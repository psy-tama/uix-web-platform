# Box

Box is the building block of most of the components. It accepts most of the css properties of CSS Box model.

```js
import { Box, Color } from 'uikit';

const CardWithTitle = () => {
  return (
    <Box
      p={2}
      width={1 / 2}
      p={4}
      mb={3}
      bg={Color.white}
      borderColor={Color.black}
      borderWidth={1}
      borderStyle="solid"
      borderRadius="10px"
    >
      <h1>This is Box example</h1>
    </Box>
  );
};
```

## Available CSS properties to use as Props

Box uses css props from `styled-system` internally. These available props can be categorized as-

- [ColorProps](https://styled-system.com/api/#color) related to Box color
- [SpaceProps](https://styled-system.com/api/#space) related to Box spacing
- [LayoutProps](https://styled-system.com/api/#layout) related to Box Layout
- [FlexboxProps](https://styled-system.com/api/#flexbox) related to Box as flex
- [BorderProps](https://styled-system.com/api/#border) related to Box borders
- [ShadowProps](https://styled-system.com/api/#shadow) related to Box shadow
- [PositionProps](https://styled-system.com/api/#position) related to Box positions
