# Color Palette

We have currently two primary colors - `black` and `white`. Below is the usage -

```js
import { Color } from 'uikit';

const ButtonsWithColor = () => {
  return (
    <>
      <Button color={Color.black} />
      <Button color={Color.white} />
    </>
  );
};
```
